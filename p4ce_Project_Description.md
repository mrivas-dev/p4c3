A mobile-first training app for CrossFit and hybrid athletes that combines structured strength progression with daily WOD training.

The app helps users improve Olympic lifts and core strength movements (snatch, clean & jerk, squat, deadlift, bench) through adaptive periodized programming, fatigue management, and lightweight workout tracking.

Core features include:
- Auto-generated 4–5 day strength cycles
- Percentage-based training using user 1RMs
- Daily readiness/fatigue scoring
- Pre-WOD strength sessions + post-WOD accessories
- Workout logging with RPE and progression tracking
- Performance analytics and trend visualization

The UX should feel fast, minimal, and training-focused — optimized for athletes who are tired after training and want quick logging without spreadsheet complexity.

Tech stack target:
- React Native / Expo mobile app
- NestJS backend
- PostgreSQL database
- Prisma ORM
- Clean modern UI inspired by premium fitness apps
- Offline-first logging with cloud sync
